## 🎉 DatarScript v0.0.1-beta.1 Released!

**"EnglishScript - Programming in Natural Language"**

Welcome to the first beta release of DatarScript, a revolutionary programming language that eliminates traditional syntax barriers by using intuitive English phrases.

### 🚀 What's New in Beta 1
- **Complete Interpreter**: Full lexer, parser, AST, and execution engine
- **Rich Builtins**: 50+ functions across math, file I/O, HTTP, JSON, graphics, terminal control
- **Natural Syntax**: Write code in plain English like "Show 'Hello World!'" or "Set x to 10 plus 5"
- **Extensive Documentation**: Professional docs with examples and tutorials
- **Cross-Platform**: Works on Linux, Windows, Mac with Python 3.8+

### 🛠️ Installation

#### From PyPI (Recommended)
```bash
pip install datarscript
```

#### From GitHub Release
Download the appropriate file for your platform and extract/install.

#### Manual Install
```bash
git clone https://github.com/ForgedInFiles/datarSCRIPT.git
cd datarSCRIPT
python3 install.py  # Sets up command-line tools
```

### 📚 Usage Examples

**Hello World**
```datarscript
Show "Hello, DatarScript!".
```

**Variables & Math**
```datarscript
Set x to 42.
Set y to x divided by 2.
Show y plus " is the answer".
```

**Functions**
```datarscript
Create a function called greet that takes name:
    Show "Hello, " plus name plus "!".
End function.

Call greet with "World".
```

**Files & I/O**
```datarscript
Write "Hello from DatarScript!" to file "output.txt".
Read file "output.txt" and store in content.
Show content.
```

### 🎯 Key Features
- ✅ Variables, arithmetic, comparisons, strings
- ✅ Lists, dictionaries, functions, recursion
- ✅ Control flow (if/else, while/for loops)

- ✅ File operations (read/write/append/delete)
- ✅ HTTP requests, JSON parsing
- ✅ Terminal control (colors, animations, cursor)
